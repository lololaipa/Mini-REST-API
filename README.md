# Mini-REST-API

Опис

У цій практичній роботі реалізовано простий REST API на Node.js для роботи з ресурсами, які зберігаються у файлі data.json.

API підтримує повний CRUD функціонал:

отримання списку
отримання елемента за id
створення нового елемента
оновлення елемента
видалення елемента
Запуск проєкту
node rest_list.js 3000
node rest_get_by_id.js 3000
node rest_create.js 3000
node rest_update.js 3000
node rest_delete.js 3000

(або запускається окремо кожен файл під час перевірки)

Структура data.json
[
  { "id": 1, "name": "Alpha" },
  { "id": 2, "name": "Beta" }
]
Реалізовані ендпоінти
GET /items

Повертає список всіх елементів
Статус: 200

GET /items/:id

Повертає один елемент за id
Статус: 200 / 404

POST /items

Створює новий елемент
Статус: 201

PUT /items/:id

Оновлює елемент за id
Статус: 200 / 404

DELETE /items/:id

Видаляє елемент за id
Статус: 200 / 404

Перевірка
eu-node-basics verify <номер> <файл>.js
